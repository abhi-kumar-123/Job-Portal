 
# 💼 Job Portal - MERN Stack Project

A full-featured **Job Portal Web Application** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)** that connects **job seekers** and **employers** on a single platform.

This platform allows employers to post jobs, manage applicants, and track job analytics, while job seekers can search jobs, apply, save opportunities, and manage their profiles.

---

# 🚀 Features

## 👨‍💼 Employer Features
- Employer Registration & Login
- Post New Jobs
- Edit / Delete Job Listings
- Manage Applicants
- View Dashboard Analytics
- Company Profile Management

## 👨‍🎓 Job Seeker Features
- Job Seeker Registration & Login
- Search Jobs with Filters
- Apply for Jobs
- Save Jobs
- Upload Resume
- Manage Profile

## 🔐 Authentication
- JWT Authentication
- Protected Routes
- Role-Based Access Control

## 📂 File Upload
- Avatar Upload
- Resume Upload
- Company Logo Upload

## 📊 Dashboard
- Employer Dashboard
- Job Statistics
- Application Tracking

# ⚙️ Installation

## Clone Repository

```bash
git clone https://github.com/your-username/job-portal.git
cd job-portal
````

---

## Install Frontend

bash
cd frontend
npm install
npm run dev


---

## Install Backend

bash
cd backend
npm install
npm run server


---

# 🔑 Environment Variables

Create `.env` file inside backend:

env
PORT=8000
MONGO_URI=your_mongodb_connection
JWT_SECRET=your_secret_key
